# 🏢 Active Directory Project

Enterprise Active Directory lab built on Windows Server 2022.
Fully automated with PowerShell.

## What the script does
- Creates OU structure (IT, Finance, Management, Users, Computers)
- Creates Security Groups (GRP-IT, GRP-Finance, GRP-Management)
- Creates 5 domain users assigned to their groups
- Installs and configures DHCP server
- Creates 6 GPOs (Password Policy, Screen Lock, Disable USB, 
  Restrict Control Panel, Wallpaper, Disable CMD)

## Tech Stack
Windows Server 2022 · Active Directory · PowerShell · DHCP · GPO
